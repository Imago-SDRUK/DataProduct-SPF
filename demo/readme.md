#### Description 
This folder contains technical stuff for the workflow development, currently with the working versions of Notebooks to produce the cloud probability product:
* **`spf.py`** - utilises `odc-stac`, `xarray` and `dask.distributed.LocalCluster`
* **`cloud_odcstac.ipynb`** – utilises `odc-stac` and `xarray` (legacy)
* **`cloud_pystac.ipynb`** – utilises `pystac-client`, `rasterio`, and `numpy` (legacy)
